What the skill does

Brand Accuracy

Exact tokens, every time

Canvas #032d42, green #63df4e, three radii, no box-shadows. Every non-obvious brand rule is encoded so Claude stops approximating and starts producing pixel-perfect output.

Two Modes

Light & dark, fully specified

Light mode for marketing and product pages. Dark mode with a four-step brightness ladder — #021a26 through #063e5a — for platform-depth interfaces. One skill, both covered.

Verification

Built-in iteration checklist

A 12-point checklist runs before Claude delivers any artifact — catching color drift, unauthorized border-radii, wrong typography weights, and missing aurora patterns before they reach you.

Built for how you actually work

Zero research overhead

No more inspecting servicenow.com, pulling hex values from DevTools, or cross-referencing spacing tokens in a separate tab. The skill carries the full design system into every Claude conversation.

Consistent across your team

One shared skill means every teammate's Claude produces the same brand-compliant output — prototypes, mockups, slide decks, and dashboards all speak the same visual language.

Progressive detail on demand

A lean 44-line SKILL.md loads the always-needed rules in milliseconds. Full typography tables, breakpoint specs, and dark-mode elevation tiers load from reference files only when a step calls for them.

How it works

1

Choose the mode

Light mode — navy-teal canvas #032d42 — is the default for marketing and product pages. Say "dark mode" or describe a platform-depth interface and Claude switches to the abyssal canvas with the full four-tier brightness ladder.

2

Apply the non-negotiable rules

Eight brand rules load immediately: pill-only buttons at 500px radius, accent-only green, three radii (16px / 24px / 500px), no box-shadows, aurora gradient on the banner only, Inter as the font substitute, weight ceiling at 700.

3

Build with ready-to-use prompts

A component library covers heroes, primary CTAs, sub-nav bars, floating FAB pills, feature cards, benefit grids, gradient-border featured cards, and dotted particle backgrounds — Claude composites them into your artifact.

4

Verify before delivery

The 12-point checklist confirms canvas color, brightness ladder, green usage, radius system, typography weights, shadow policy, aurora placement, floating FABs, and sub-nav active-tab underlines before Claude hands off the output.
